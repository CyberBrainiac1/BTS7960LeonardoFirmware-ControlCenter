Place avrdude.exe and avrdude.conf in this folder.

Recommended source (Windows):
- Arduino IDE installation: <Arduino>/hardware/tools/avr/bin/avrdude.exe
- Arduino IDE installation: <Arduino>/hardware/tools/avr/etc/avrdude.conf

The Control Center will look for:
- Assets/Tools/avrdude/avrdude.exe
- Assets/Tools/avrdude/avrdude.conf
